# Blender 快捷键记录

|快捷键|功能|
|:---:|:---:|
|`Shift` + `c`|cursor复原|
|`z`|显示线框|
|`Tab`|编辑模式和物体模式切换|
|`a`|全选、取消全选|
|`Shift` + `a`|添加|
|`m`|将选中物体移动到某一层|
|`b`|线框选择|
|`Ctrl` + `左键`|自由选择|
|`Ctrl` + `u`|保存启动文件|
|`Ctrl` + `↑` 或 `Shift` + `Space`|最大化某一窗口|
|`t`|展开、隐藏左侧工具列|
|`n`|展开、隐藏右侧工具列|
|`Ctrl` + `Tab`|切片点、线、面选择|
|`e`|整体挤出|
|`g`|移动|
|`r`|旋转|
|`s`|缩放|
|`x`|删除|
|`Ctrl` + `r`|环切并滑动|
|`F12`|渲染场景|
|`k`|刀片工具|
|`f`|快速建立表面|
